# lab-a09-operators

**Operators on DCS** — a DCS Academy Foundations workshop.

Understand the OpenShift Operator pattern (controller + reconcile), CRD vs CR, and OLM/
OperatorHub — then create a CloudNativePG `Cluster` Custom Resource and watch the operator
reconcile it. The core takeaway is the DCS ownership model: the platform owns the operator;
the tenant owns the application instance it manages.

Part of Module A (Foundations). Prerequisite: lab-a02-kubernetes-essentials. Prerequisite
for Module F (Operators track).
