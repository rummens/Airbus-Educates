# lab-a04-harbor-registry

**Working with Harbor** — Foundations (Module A, core) workshop for the DCS Academy.

Learners use the DCS Harbor registry: pull an image from a catalog with `skopeo`,
run it on the cluster, browse the Harbor UI (projects, tags, digests, scan results),
and see how vulnerability scanning gates images. Pull-only — pushing is covered as a
concept. Prerequisite: lab-a02-kubernetes-essentials.

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-a04-harbor-registry.md`.
