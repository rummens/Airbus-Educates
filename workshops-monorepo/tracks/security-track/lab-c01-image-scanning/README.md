# lab-c01-image-scanning

**Image Scanning & Harbor Gates** — Security & Compliance (Module C) workshop for the DCS Academy.

Learners read Harbor vulnerability and compliance scan results with `jq`, understand the
**scan gate** that blocks unsafe images at pull time, remediate a flagged image (patched tag
or rebuild on a patched base), and learn the **Security Exception Process** (ITSM) escape
hatch. Scan reports are static fixtures so every step is examiner-verifiable air-gapped.
Prerequisite: lab-a04-harbor-registry.

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-c01-image-scanning.md`.
