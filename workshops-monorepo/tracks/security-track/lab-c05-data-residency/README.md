# lab-c05-data-residency

**EU Data-Residency & Compliance** — Security & Compliance (Module C) elective workshop for
the DCS Academy.

Learners read the DCS **data-classification** scheme and multi-national (e.g. Germany/Spain)
**residency** guarantees, see how workload **placement** is expressed (region labels +
`nodeSelector`) and inspect it, read the **Responsibility Matrix (RACI)** to tell platform
duties from tenant duties, and walk the **Security Exception** / **ITSM** compliance loop a
tenant follows. Observe/concept-heavy: governance artefacts are read-only fixtures. Assumes
Module A (tenancy).

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-c05-data-residency.md`.
