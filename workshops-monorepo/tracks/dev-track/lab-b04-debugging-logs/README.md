# lab-b04-debugging-logs

**Debugging & Logs** — Developer (Module B, dev track) workshop for the DCS Academy.

The sample app is pre-deployed **broken** (an unmirrored image tag → `ImagePullBackOff`).
Learners work the **observe → hypothesise → fix → verify** loop with `oc get`,
`oc describe`, `oc get events` and `oc logs`, find the root cause, fix it, and confirm
recovery. Prerequisite: lab-b01 (Deploy Your First App).

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-b04-debugging-logs.md`.
