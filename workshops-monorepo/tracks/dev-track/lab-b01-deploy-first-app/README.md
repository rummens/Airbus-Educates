# lab-b01-deploy-first-app

**Deploy Your First App on DCS** — Developer (Module B, dev) workshop for the DCS Academy.

Learners take the `hello-dcs` sample app from a Harbor image to a running, reachable
workload in their DEV namespace: deploy it with a Deployment, give it a stable address with
a Service, expose it through the Educates session ingress, and iterate on it. Prerequisite:
Module A (Foundations) — especially A02, A03, A06.

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-b01-deploy-first-app.md`.
