# lab-a06-networking

**Networking & Exposing Apps** — a DCS Academy Foundations workshop.

Expose an application on DCS via the session proxy and an OpenShift Route, learn the DCS
traffic chain (Service → Route → External Load Balancer with DCS-managed DNS), and see how
Network Policies (observe-only) and air-gapped egress restrictions shape traffic.

Part of Module A (Foundations). Prerequisite: lab-a02-kubernetes-essentials.
