# lab-a03-namespace-model

**Namespaces & the Prod/Dev Model** — a DCS Academy Foundations workshop.

Understand the DCS DEV and PROD namespace types, see them side by side in a per-session
virtual cluster, and learn why work is promoted from DEV to PROD rather than edited in
place. The concrete DEV-vs-PROD difference is demonstrated live: PROD enforces a Kyverno
admission policy, DEV does not.

Part of Module A (Foundations). Prerequisite: lab-a02-kubernetes-essentials.
