# lab-a07-storage

**Storage on DCS** — Foundations (Module A, core) workshop for the DCS Academy.

Learners request persistent storage with a PersistentVolumeClaim, learn the
PVC → StorageClass → PV model, compare File (RWX) and Block (RWO) storage,
mount a volume into the A02 sample app, and prove data survives a pod restart.
The lab also covers how data classification drives storage-class choice and how
S3 object storage is obtained via an ITSM ticket. Prerequisite: lab-a02-kubernetes-essentials.

Built with the `airbus-educates-workshop-authoring` skill. See the plan at
`planning/workshop-plans/lab-a07-storage.md`.
